# Image Uploader


## what
一款`chrome`插件，可以将你电脑上的一张图片变成一个url访问。

## features 
1. 上传本地图片到服务器，得到图片url
2. 支持拖拽上传
3. 记录上传历史的图片的URL
4. 支持复制

![](./docs/screenshot.png)

## why
因为经常用`markdown`写东西，用`![]()`引用图片，没有找到一个合适的传图工具，于是自制一个。

最主要的原因是还是用了Mac，不然可以用之前写的windows的版本，
详见： [博客园图床ImageBed](https://github.com/FrankFan/CnblogsImageBed)
